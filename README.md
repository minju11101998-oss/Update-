# ⚔️ ARENA BATTLE — Telegram PvP RPG Bot

## Установка

```bash
pip install python-telegram-bot==20.7 aiosqlite
```

## Настройка

1. Создай бота у [@BotFather](https://t.me/BotFather)
2. В `config.py` замени:
   ```python
   BOT_TOKEN = "ВАШ_ТОКЕН"
   ADMIN_IDS = [ВАШ_USER_ID]
   ```
3. Запуск:
   ```bash
   python bot.py
   ```

## Игровые механики

### Классы
| Класс | ✊ Сила | ♥️ Выносливость | 💫 Ловкость | 🧿 Интуиция | Умение |
|---|---|---|---|---|---|
| ⚔️ Воин | 8 | 7 | 4 | 3 | Блок (25% блок 60% урона) |
| 🗡️ Ассасин | 6 | 4 | 9 | 5 | Смертельный удар (+50% при крите) |
| 🔮 Маг | 5 | 4 | 5 | 10 | Предчувствие (+30% предвидение) |
| 💢 Берсерк | 11 | 5 | 5 | 2 | Ярость (Атака +40%, Защита -20%) |

### Формула статов из навыков
```
❤️ HP       = 80  + Выносливость × 12
🗡 Атака    = 10  + Сила × 3
🔰 Защита   = 5   + Выносливость × 2
🥊 Крит%    = 3   + Ловкость × 1.2
💥 СилаКрит = 140 + Сила × 2 (%)
👁 Предвид  = 2   + Интуиция × 1.5 (снижение урона)
⚡️ Уворот   = 2   + Ловкость × 1.3 (%)
🤺 Контруд  = 1   + Интуиция × 0.8 (% урон назад)
🎯 Точность = 85  + Ловкость × 0.5 (%)
```

### Механика боя
- Пошаговый бой, оба игрока выбирают действие одновременно
- **⚔️ Атака** — стандартный удар
- **✨ Умение** — классовое умение
- **🛡️ Защита** — снижает входящий урон на 40% (воин — 60% с шансом 25%)
- **💊 Зелье** — восстанавливает 25% макс. HP
- Если игрок не ответил за 45 секунд — автоматически выбирается Атака
- При ничьей (оба умирают) — ничья

### Матчмейкинг
1. Игрок нажимает `/arena`
2. 30 секунд поиска реального противника
3. Если не нашли — бой с ботом (подобранным под уровень)

## Команды игрока

| Команда | Действие |
|---|---|
| `/start` | Регистрация, выбор класса |
| `/profile` | Профиль, статы, экипировка |
| `/arena` | Поиск боя |
| `/equipment` | Магазин экипировки |
| `/top` | Таблица лидеров |
| `/cancel` | Отменить поиск |

## Команды администратора

| Команда | Пример |
|---|---|
| `/admin` | Открыть панель |
| `/give_gold [id] [amount]` | `/give_gold 123456 500` |
| `/ban [id]` | `/ban 123456` |
| `/unban [id]` | `/unban 123456` |
| `/set_level [id] [lvl]` | `/set_level 123456 10` |
| `/broadcast [текст]` | `/broadcast Ивент начался!` |

## Расширение

### Добавить предмет
```python
# в gamedata.py → ITEMS
"thunder_axe": {
    "name": "⚡ Громовой топор",
    "slot": "weapon",
    "desc": "Бьёт молнией.",
    "req_level": 15,
    "price": 800,
    "stats": {"atk": 40, "crit_chance": 10, "crit_power": 35},
```
